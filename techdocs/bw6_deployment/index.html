TIBCO BW6 build, deploy und start eines EARS

Aug 25, 2017

Nachdem wir unseren Prozess erstellt haben muss mit dem Designer das EAR File erzeugt werden. Dies geht entweder direkt im Designer oder über die Kommandozeile

./bwdesign -data /home/tibco/tibworkspace/sample system:validate
./bwdesign -data /home/tibco/tibworkspace/sample system:export -ear sample.app /home/tibco/

Anschließend können wir mit dem bwadmin die Applikation deployen und starten:

./bwadmin

upload -d myDBDomain /home/tibco/sample.app_1.0.0.ear
deploy -d myDBDomain -a myAppspace  sample.app_1.0.0.ear
start -d myDBDomain -a myAppspace application sample 1.0

Die Befehle können dem bwadmin auch über ein Answerfile übergeben werden:

printf "upload -d myDBDomain /home/tibco/sample.app_1.0.0.ear\ndeploy -d myDBDomain -a myAppspace  sample.app_1.0.0.ear\nstart -d myDBDomain -a myAppspace application sample 1.0" > bw6setup.txt

./bwadmin -f /home/tibco/bw6setup.txt
