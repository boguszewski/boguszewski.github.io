TIBCO BW5 Export und Import von Tibco Domain Usern

Jan 16, 2016

Die User einer Tibco BW5 Domain können über die Kommandoziele im- und exportiert werden. Das Tool ExportDomainSecurity im Verzeichnis <TIBCO_HOME>/tra/<VERSION>/bin hilft dabei

./ExportDomainSecurity -domain test_avb -allUsers -roleMembership -ACLS -GUID -cred credentials -file /tmp/users

ausführen um eine Datei mit Nutzern zu exportieren.
