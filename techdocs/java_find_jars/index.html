Dec 12, 2015

Java Klassen in Jars finden Um um herauszufinden in welchem JAR Archiv sich eine Java Klasse versteckt, kann man unter Linux die Kombination von jar. grep, and find verwenden (Achtung case sensitively)

find path/to/libs -name '*.jar' -exec grep -Hls ClassName {} \;
